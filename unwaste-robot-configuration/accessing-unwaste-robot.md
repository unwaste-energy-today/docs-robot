# Accessing Unwaste Robot

You can access an **Unwaste Robot** in three ways:

* locally on your LAN,
* through Unwaste Cloud Web,
* through Unwaste Cloud Mobile.

Each access method serves a different purpose.

| Access method        | Where you access it                                                            | Primary use                                                                                                                                         |
| -------------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Local access         | On the same LAN as the Unwaste Robot.                                          | Initial configuration of devices and system functions, plus system updates. Usually performed by an installer.                                      |
| Unwaste Cloud Web    | In a web browser [https://cloud.unwaste.energy](https://cloud.unwaste.energy/) | Configure the electrical installation using added devices and entities. Change advanced system settings. Usually managed by a system administrator. |
| Unwaste Cloud Mobile | In the mobile app that is available in Google Play and Apple App Store         | Daily user system operation, device mode changes, monitoring, alerts, data analysis, and recommendations.                                           |
|                      |                                                                                |                                                                                                                                                     |
