# Installation options

Unwaste Robot can be installed in two ways:

1. As a Home Assistant application, formerly called an add-on. This option is available for users with Free or Plus licenses. See [Installation for Home Assistant](https://app.gitbook.com/s/Wf3M0a04WuhvfEbAgXFl/installation-for-home-assistant).
2. On a dedicated Unwaste OS system. This option is for all users. See [Installation for Unwaste OS](https://app.gitbook.com/s/Wf3M0a04WuhvfEbAgXFl/installation-for-unwaste-os) for more information.

Unwaste OS is the recommended option for production deployments. It provides more reliable operation and stronger security.
