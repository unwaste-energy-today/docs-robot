# Installation options

Unwaste Robot can be installed in two ways:

1. As a Home Assistant application, formerly called an add-on. This option is available for users with Free or Plus licenses. See [Installation for Home Assistant](../unwaste-for-home-assistant/installation.md).
2. On a dedicated Unwaste OS system. This option is available for users with any type of Unwaste license. See [Installation for Unwaste OS](../unwaste-os/installation.md) for more information.

Unwaste OS is the recommended option for production deployments. It provides more reliable operation and stronger security.
