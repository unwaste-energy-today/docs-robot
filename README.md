# Welcome

Welcome to the Unwaste EMS documentation.

Unwaste is an energy management system for optimizing consumption, production, and storage.

## Start here

1. Review [Getting started](<Getting started/Introduction.md>) and choose your installation method.
2. Install [Unwaste Robot for Home Assistant](unwaste-for-home-assistant/installation.md) or [Unwaste Robot for Unwaste OS](unwaste-os/installation.md).
3. Model your electrical installation in [Configuration basics](unwaste-robot-configuration/configuration.md).
4. Use and monitor the system through [System Operation](unwaste-robot-operation/system-operation.md).

## Explore the documentation

Start with [Unwaste EMS](<Introduction/About Unwaste EMS.md>) to understand the system.

Then use these sections when needed:

* [Configuration](unwaste-robot-configuration/configuration.md) — model connections, circuits, production, storage, and devices.
* [Profiles, schedules and overrides](unwaste-robot-configuration/profiles-schedules-and-overrides/profiles-schedules-and-overrides.md) — control device behavior.
* [Unwaste Cloud](unwaste-cloud/benefits-of-cloud-account.md) — use cloud services and remote access.
