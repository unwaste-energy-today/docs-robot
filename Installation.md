# Installation

This document describes the installation process for the unWaste.energy system.

## System Requirements

* Home Assistant environment
* Raspberry Pi 4 or x86_64 architecture
* Minimum 2GB RAM
* Network connectivity for data collection

## Installation Steps


1. **Prepare Home Assistant Environment**
   * Ensure Home Assistant is running and accessible
   * Verify supervisor mode is enabled
2. **Install unWaste.energy Add-on**
   * Add the unWaste.energy repository to Home Assistant
   * Install the add-on through the supervisor interface
3. **Initial Configuration**
   * Configure API endpoints
   * Set up device integrations
   * Define energy tariff structure
4. **Verification**
   * Check system status
   * Verify data collection
   * Test control mechanisms

## Post-Installation

After successful installation, proceed to system configuration to set up your specific energy management setup.
